# Expense Tracker

A command-line expense tracker built with Python.

This project allows users to record, organize, and analyze their daily expenses through a simple terminal interface.

## Features

- Add new expenses
- View all expenses
- Categorize expenses
- Calculate total spending
- Store data locally
