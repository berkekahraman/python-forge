# Expense Tracker

A command-line expense tracker built with Python.

This application allows users to manage their daily expenses through a simple terminal interface. All data is stored locally in a JSON file.

## Features

- Add new expenses
- View all expenses
- Delete expenses
- Filter expenses by category
- Display category summary
- Calculate total spending
- Save and load expenses using JSON
- Input validation

## Technologies

- Python 3
- JSON
- Git
- GitHub
